Rocket Fireworks Celebration Plugin for your blog or website.

== Description ==
<p>Rocket Fireworks Celebration Plugin for your blog or website.</p>
<p>Celebrate with cool Rocket Firework show for your website or blog visitors. This Plugin is very easy to install & works as great tool when you want celebrate important days with your website visitors by giving them pleasant user experience. Firework rockets blows in multicolor which looks cool, attractive & attentive. This Plugin is perfect match made when you are in mood of celebration with your customers.</p>
<p>This Plugin is very useful for Diwali, Christmas and New year celebrations.</p>
<p>Just activate the Plugin it will start rocket fireworks. Enjoy your best days.</p>

== Installation ==
1. Upload the plugin files to the '/wp-content/plugins/rocket-fireworks' directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. After activating the plugin rocket fireworks starts.
